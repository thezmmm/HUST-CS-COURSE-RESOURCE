## 5

| 前缀匹配          | 链路接口 |
| ----------------- | -------- |
| 11100000 00       | 0        |
| 11100000 01000000 | 1        |
| 1110000           | 2        |
| 11100001 1        | 3        |
| other             | 3        |

- 第一个数据开头是110 直接匹配第五个，链路接口3
- 第二个数据 1110000匹配第三个，链路接口2
- 第三个数据 11100001 1 匹配第四个，链路接口3
