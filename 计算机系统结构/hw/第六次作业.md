## 10.6

远程访问率 p = 0.5%,远程访问时间 t = 2000ns,时钟周期 T = 10ns

远程访问开销 C = t/T = 2000ns/10ns = 200

有 0.5%远程访问的机器的实际 CPI2 为:

CPI2 = CPI1 + p×C = 1.0 + 0.5%×200 = 2.0

只有局部访问的机器的基本 CPI1 = 1.0

CPI2/ CPI1 = 2.0/1.0 = 2

因此,没有远程访问状态下的机器速度是有0.5% 远程访问的机器速度的2 倍
